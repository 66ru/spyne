Protocols¶

The rpclib.protocol package contains the implementations of various remote procedure call protocols.

Protocol Base Class¶

This module contains the ProtocolBase abstract base class for all protocol implementations.

class rpclib.protocol._base.ProtocolBase(app=None, validator=None)[source]¶

This is the abstract base class for all protocol implementations. Child classes can implement only the required subset of the public methods.

The ProtocolBase class supports the following events:

  • before_deserialize: Called before the deserialization operation is attempted.
  • after_deserialize: Called after the deserialization operation is finished.
  • before_serialize: Called before after the serialization operation is attempted.
  • after_serialize: Called after the serialization operation is finished.

HttRpc¶

This module contains the HttpRpc protocol implementation.

class rpclib.protocol.http.HttpRpc(app=None, validator=None)[source]¶

The so-called ReST-minus-the-verbs HttpRpc protocol implementation. It only works with the http server (wsgi) transport.

It only parses GET requests where the whole data is in the ‘QUERY_STRING’.

The rpclib.application.Application instance this protocol belongs to.
