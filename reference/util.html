Model Utilities¶

This module contains the classes and methods for converting and validating rpclib.model.ComplexModel instances

class rpclib.util.model_utils.ComplexModelConverter(model_instance, tns, include_parent=False, parent_tag='root', include_ns=True)¶

A class to handle exporting a ComplexModel to different representations

Currently supported export targets are lxml.etree.Element, string and xml documents.

This functionality will most likely be moved into the ComplexModel itself if it proves useful and there is a willingness to modify the current ComplexModel API.
