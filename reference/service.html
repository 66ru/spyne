Service Definition¶

The rpclib.protocol package contains the implementations of various remote procedure call protocols.

Service Base Class¶

class rpclib.service.ServiceBase[source]¶

This class serves as the base for all service definitions. Subclasses of this class will use the srpc decorator or its wrappers to flag methods to be exposed.

It is a natural abstract base class, because it’s of no use without any method definitions, hence the ‘Base’ suffix in the name.

classmethod call_wrapper(ctx)[source]¶

Called in place of the original method call. You can override this to do your own exception handling.

Parameters:ctx – The method context.
